# 🎬 Movie Recommendation System

## 📌 Overview
This project is a **Movie Recommendation System** built using **Machine Learning (TF-IDF + Cosine Similarity)**.  
It suggests similar movies based on content like overview, genre, and tagline.

## 🧠 How It Works
1. Combine movie features → overview + genres + tagline
2. Convert text → TF-IDF vectors
3. Compute similarity using cosine similarity
4. Recommend top similar movies
